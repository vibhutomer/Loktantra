# Loktantra Client

Frontend application for the Loktantra blockchain voting system.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features

- Login/Connect Wallet page
- Voter Dashboard with candidate cards
- Vote confirmation modal
- Admin panel for election management
- Live results page
- Voting history and audit logs

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
